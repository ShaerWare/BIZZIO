**Контекст:** B2B-платформа для строительной индустрии (Laravel 12 + Orchid + PostgreSQL)
**Статус:** Завершено 9/10 спринтов (90% MVP), бэклог 53/75 задач (71%)
**Окружение:** Claude Code (локальная разработка, Docker)

## Быстрая справка
- Все команды в Docker: `docker compose exec app php artisan <command>`
- Тесты: `docker compose exec app php artisan test` (185 тестов, SQLite in-memory)
- Dev-режим: `composer run dev` (Vite + Queue + Logs)
- Документация в `docs/`, отчёты в `docs/sprints/`

## Важные паттерны
- PostgreSQL поиск: `ilike` (не `like`), с fallback на `like` для SQLite тестов
- Quick search API (`/search/quick`): возвращает плоский JSON-массив
- Терминология: «Запрос цен» (не «Запрос котировок»)
- Events: зарегистрированы в `AppServiceProvider@registerEventListeners()`
