# Проект Bizzo.ru — Продолжаем разработку

**Контекст:** B2B-платформа для строительной индустрии (Laravel 12 + Orchid)
**Статус:** Завершено 8/10 спринтов (80% MVP), ~360 часов разработки
**Окружение:** Claude Code (локальная разработка, Docker)

## Быстрая справка
- Все команды в Docker: `docker exec my_project_app php artisan <command>`
- Тесты: `docker exec my_project_app php artisan test`
- Документация в `docs/`, отчёты в `docs/sprints/`

## Завершённые спринты
1. Инфраструктура + Авторизация
2. Модуль "Компании"
3. Модуль "Проекты"
4. Модуль "Запрос котировок" (RFQ)
5. Модуль "Аукцион"
6. Модуль "Новости" (RSS)
7. Лента активности + Уведомления
8. Поиск + Фото (Laravel Scout, аватары, галерея компаний)
