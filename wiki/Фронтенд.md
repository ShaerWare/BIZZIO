# Фронтенд

## Стек

| Технология | Назначение |
|------------|-----------|
| Laravel Blade | Серверный рендеринг шаблонов |
| Tailwind CSS (CDN) | Утилитарные классы стилей |
| Alpine.js 3.x | Клиентская интерактивность |
| Axios | HTTP-запросы с CSRF-токеном |
| Custom CSS | Стилизация landing page |
| Custom JS | Landing page интерактивность |

## Layouts

### app.blade.php (Authenticated)
Основной макет для авторизованных пользователей:
- CDN: Tailwind CSS, Alpine.js
- Навигация (компонент)
- Опциональный header slot
- Flash-сообщения (success/error)
- Yandex.Metrika (production only)
- `@stack('styles')`, `@stack('scripts')` для страничных ресурсов

### guest.blade.php (Auth Pages)
Минимальный макет для страниц аутентификации:
- Центрированная карточка
- Логотип + форма

### navigation.blade.php
Навигационная панель:
- **Desktop:** горизонтальное меню с dropdown'ами
- **Mobile:** бургер-меню
- **Глобальный поиск:** Alpine.js с дебаунсом 300мс
- **Уведомления:** колокольчик с badge непрочитанных
- **Dropdown'ы:** "Закупки", "Новости", профиль пользователя

## Landing Page (welcome.blade.php)

Стилизация в `public/css/custom.css` (619 строк):
- Full-height hero с видео-фоном
- Тёмный overlay (opacity 0.6)
- Split layout: features (gradient) + login form (white)
- Градиент: `#28a745 → #81b407`
- OAuth кнопки: Google (красная) + Yandex (зелёная)
- Адаптивность: 991px, 768px, 576px breakpoints
- Анимации: fadeIn (0.6s), hover translateY

## JavaScript

### resources/js/app.js
Bootstrap Alpine.js + Axios с CSRF-токеном.

### public/js/custom.js (123 строки)
Landing page:
- Мобильное меню (бургер с анимацией)
- Автовоспроизведение видео
- Smooth scroll
- Автофокус первого input
- Клиентская валидация форм

### Alpine.js Patterns
- **Post form:** превью изображений перед загрузкой
- **Модальные окна:** focus management, Escape to close
- **Dropdown:** click-away to dismiss
- **Search:** debounced AJAX, результаты в dropdown
- **Auction trading:** long-polling обновление состояния
